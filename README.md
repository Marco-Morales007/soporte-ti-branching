# Help Desk | Soporte Técnico

Repositorio del área de **Soporte TI (Help Desk)** que centraliza documentación, procesos, scripts y base de conocimiento para la gestión eficiente de incidencias y solicitudes de servicio.Nuestro equipo esta conformado por:
- Hernandez Velasco Dayana
- Amy Lisette Garcia Perez
- Marco Morales
Este equipo esta enfocado en la resolucion efectiva de problemas y atencion de usuario.
---

## Objetivo

Brindar soporte técnico ágil, eficiente y orientado al usuario, garantizando la continuidad operativa mediante la resolución de incidencias y la mejora continua del servicio.

---

## Contenido del repositorio

El equipo de Help Desk cubre:

- Soporte de primer nivel (diagnóstico y resolución inicial)
- Escalamiento a niveles superiores (Nivel 2 / Nivel 3)
- Soporte a sistemas operativos (Windows, macOS, Linux)
- Gestión de usuarios y accesos 
- Soporte de red y conectividad
- Instalación y configuración de software y hardware
- Soporte por medio de correo, videollamadas y llamadas
- Gestión de tickets
